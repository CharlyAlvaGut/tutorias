Plataforma web para la gestión de tutorías académicas y administración documental mediante FlowDocs. Permite programar sesiones, colaborar en documentos, realizar comunicación interna y automatizar flujos de trabajo educativos.

Funcionalidades Principales:
- Gestión de usuarios: registro, inicio de sesión y perfiles por rol (tutor, tutorado, administrador).
- Programación de tutorías: agenda, confirmaciones, cancelaciones y notificaciones automáticas.
- Comunicación interna: chat y videollamadas mediante tecnologías web.
- FlowDocs: carga, descarga y clasificación de documentos, control de versiones y coedición colaborativa.
- Automatización de flujos: revisiones, aprobaciones y recordatorios.
- Seguridad y control de accesos según rol.
- Panel de seguimiento y reportes académicos.

Tecnologías Utilizadas:
- Backend: Spring Boot
- Frontend: Thymeleaf
- Base de datos: MySQL
- Módulo documental: FlowDocs Engine

Metodología de Desarrollo:
El proyecto se desarrolla con la metodología ágil Scrum, mediante iteraciones cortas (Sprints) e incrementos funcionales continuos.

Roadmap General:
- Autenticación y gestión de usuarios
- Módulo de tutorías
- Comunicación en tiempo real
- Integración de FlowDocs
- Automatización de flujos
- Panel de seguimiento y reportes
